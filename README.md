# MINDSGGER

This is an app that provides information about the activities of a device (specifically the keyboard).

Pre-requisite:
- Create an App Gmail password

## Steps just for Linux users:

###### 1. Enable execute permission
```SHELL
chmod +x run.bash
```
###### 2. Reload (Optional)
```SHELL
source ./run.bash
```
###### 3. Run
```SHELL
./run.bash
```
